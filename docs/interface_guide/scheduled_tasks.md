# スケジュールされたタスク

ウェブ管理パネルの左側 -> システム -> スケジュールされたタスク

スケジュール・タスクでは、LEDライトのオン／オフ、ルーターの再起動、Wi-Fiのオン／オフなど、基本的な動作の1日のスケジュールを設定できます。

**注意**： この機能を使用する前に、まず[Time Zone](time_zone.md) で時刻を同期させてください。設定した時刻にデバイスがシャットダウンしている場合、タスクは実行されません。
