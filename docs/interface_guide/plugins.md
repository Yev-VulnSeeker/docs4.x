# プラグイン

ウェブ管理画面の左側 -> アプリケーション -> プラグイン

**プラグイン**では、OpenWrtのパッケージを管理することができます。任意のパッケージをインストールしたり、削除したりすることができます。

使用する前に、**アップデート**ボタンをクリックすることをお勧めします。

下図はGL-AXT1800のプラグインページです。
